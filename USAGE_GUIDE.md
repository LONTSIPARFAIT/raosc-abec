# 📖 Guide d'Utilisation de la Plateforme RAOSC

Ce document explique comment utiliser les différentes fonctionnalités de la plateforme pour les Administrateurs et les Organisations.

---

## 🎖️ Pour l'Administration (ABEC)

L'administration a pour rôle de superviser les demandes et d'assurer la qualité des données sur l'annuaire.

### 1. Accès au Tableau de Bord
Connectez-vous avec un compte ayant le rôle `admin`. Vous atterrirez sur un dashboard spécialisé affichant :
- Le nombre total d'OSC approuvées.
- Le nombre de demandes en attente de validation.
- La liste des notifications système.

### 2. Valider une Organisation
Les nouvelles demandes apparaissent dans la section **"Demandes d'inscription"**.
- **Approuver :** Cliquez sur le bouton "Check" (vert). L'organisation sera immédiatement publiée sur l'annuaire public.
- **Rejeter :** Cliquez sur le bouton "X" (rouge). Un champ s'ouvre pour saisir le **motif du rejet**. Une notification (Portail + Email) sera envoyée au créateur de l'organisation.

---

## 🏢 Pour les Organisations (OSC)

### 1. Inscription
- Cliquez sur **"Inscrire mon ONG"** depuis l'accueil ou le dashboard.
- Remplissez le formulaire avec des informations précises (Nom, Localisation, Secteur d'activité, N° d'enregistrement).
- Une fois soumis, votre statut sera **"En attente"**.

### 2. Gestion du Profil
Depuis votre dashboard personnel, vous pouvez :
- Voir le statut de validation de votre structure.
- Accéder à votre **Médiathèque** (en cours de développement) pour gérer les images de votre galerie.
- Cliquer sur **"Voir mon profil public"** pour voir le rendu final sur l'annuaire.

### 3. Galerie & Médias
Les organisations approuvées peuvent mettre en avant leurs actions grâce à une galerie d'images. Ces images apparaissent dans un slider premium sur leur page de détail publique.

---

## 🔍 Utilisation de l'Annuaire Public

Tout utilisateur (même non connecté) peut consulter l'annuaire :
- **Recherche Instantanée :** Tapez le nom d'une organisation dans la barre de recherche. Les résultats s'actualisent sans recharger la page.
- **Filtres Avancés :** Filtrez par **Catégorie** (Environnement, Santé, Éducation, etc.) ou par **Ville**.
- **Page de Détail :** Cliquez sur une organisation pour voir son profil complet, sa galerie photo, et ses coordonnées de contact.

---

## 🔔 Notifications
- Restez informé via l'icône **cloche** sur votre dashboard.
- Les changements critiques (Approbation/Rejet/Suspension) vous sont également envoyés par **email**.
